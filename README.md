torrent_cli
 
 The compleate solution for torrent cli magnet download 

Installation Guide

Step 1
        clone the repository

        $ git clone https://github.com/nithinlycanz/torrent_cli.git

Step 2
        change directory

        $ cd torrent_cli

Step 3
        install npm packages

        $ npm install

Step 4
        run it

        $ node torrent.js <Magnet Link>
